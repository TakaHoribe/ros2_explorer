# ros2_explorer

For those who frequently use ros2 node info, topic info, echo, hz, param, etc.

## install and run

```sh
pip install -r requirements.txt
```

```sh
python3 ./script/ros2_explorer.py
```

Then access to `http://127.0.0.1:8050/` in your browser.
